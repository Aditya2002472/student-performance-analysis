# Student Performance Data Analysis

## Project Overview
This project analyzes student academic performance using Python and basic data analytics techniques. The goal is to identify patterns in student scores based on study hours, gender, and parental education.

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel

## Key Steps
1. Data Loading
2. Data Cleaning (missing values and duplicates)
3. Feature Engineering (Total Score calculation)
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Exporting processed results

## Key Insights
- Students who study more hours generally achieve higher scores.
- Parental education level shows some correlation with student performance.
- Score distribution shows most students fall in a mid-performance range.

## Files
- `student_analysis.py` → main analysis script
- `student_data.csv` → dataset
- `cleaned_student_data.xlsx` → processed data
- `gender_analysis.csv` → gender-based analysis
- `parent_education_analysis.csv` → parental education analysis
